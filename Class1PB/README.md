# 📊 Power BI Charts – Bar & Column Visualizations

## 📌 Overview
This project demonstrates different types of bar and column charts in Power BI.

---

## 📊 Chart Types

### 🔹 Clustered Bar Chart
**Definition:** Displays data using horizontal bars side-by-side.

**When to Use:**
- Compare multiple categories
- Long category names

![Clustered Bar Chart](./Class1PB/Clustured Bar.JPG)

---

### 🔹 Clustered Column Chart
**Definition:** Displays vertical bars for comparison.

**When to Use:**
- Compare values across categories
- Show trends

![Clustered Column Chart](./clustered-column.png)

---

### 🔹 Stacked Bar Chart
**Definition:** Shows total with category contributions (horizontal).

**When to Use:**
- Part-to-whole analysis
- Compare totals + segments

![Stacked Bar Chart](./stacked-bar.png)

---

### 🔹 Stacked Column Chart
**Definition:** Vertical stacked representation.

**When to Use:**
- Track composition over time

![Stacked Column Chart](./stacked-column.png)

---

### 🔹 100% Stacked Bar Chart
**Definition:** Shows percentage contribution.

**When to Use:**
- Compare proportions only

![100% Stacked Bar Chart](./100-stacked-bar.png)

---

### 🔹 100% Stacked Column Chart
**Definition:** Vertical percentage distribution.

**When to Use:**
- Compare percentage trends

![100% Stacked Column Chart](./100-stacked-column.png)

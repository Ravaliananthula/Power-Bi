# 📊 Power BI Charts – Bar & Column Visualizations

## 📌 Overview
This project demonstrates different types of bar and column charts in Power BI.

## 📊 Types of Charts

### 🔹 Clustered Bar Chart
**Definition:** Displays data using horizontal bars side-by-side.

**When to Use:**
- Compare multiple categories
- Long category names
